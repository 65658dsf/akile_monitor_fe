Akile Monitor 前端开源项目-Stellar修改版，各位请自行下载源代码后构建项目
本版在原有的基础上增加了卡片/列表视图切换，按需排序，点击网络情况卡片可以展开排行榜

## 安装
```bash
git clone https://github.com/65658dsf/akile_monitor_fe.git

cd akile_monitor_fe

npm install
```

## 配置文件
```
构建后config.json中进行修改
```

## 打包
```bash
npm run build
```

## 开发
```bash
npm run dev
```
